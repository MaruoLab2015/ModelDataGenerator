# ModelDataGenerator

ModelDataGenerator provide to make scan data of a ring or overhang model base 

## Ring data

```bash
$ cd ModelDataGenerator-master
$ javac createRing.java 
$ java createRing

周回数 : 3
ファイルの作成に成功

・
```
